# README

This is a Ruby on Rails CRUD with Trailblazer architecture

* Ruby version
    ````
    2.7.5
    ````
* System dependencies
    ````
    trailblazer-rails
    slim-rails
    reform-rails
    sidekiq
    ````
* Database creation
    ````
    rails db:create
    rails db:migrate
    ````

* Database initialization
    ````
    rails db:seed
    ````
* Run
    ```
    rails s
    ````
